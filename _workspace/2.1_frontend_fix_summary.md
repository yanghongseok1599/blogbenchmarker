## 미해결 / 후속 의존

1. **E-1 로그아웃**: `background/handlers/auth-handler.js` + `mypage/*.js` 에서 구현 필요(Phase 2.2 / scope 타 에이전트).
2. **manifest `identity` 권한**: `chrome.identity.launchWebAuthFlow` 동작을 위해 `manifest.json permissions`에 `"identity"` 추가 필요. 본 작업은 manifest 수정 금지 규칙으로 TODO 주석만 기록(login.js:107).
3. **`lib/supabase-client.js` 실파일 생성**: 기획자 동시 작업분. 생성 완료 전에는 `import` 해석 실패로 3개 JS 파일이 로드 불가. 기획자 커밋 후 통합 테스트 필요.
4. **`auth/callback.html`**: OAuth redirect landing. 현재는 `chrome.identity.getRedirectURL()` 경로 사용으로 별도 HTML 없이 동작 가능하나, 추후 UX 개선 시 추가 검토.
5. **`_locales/ko/messages.json`**(HIGH A-4): Phase 10 i18n 작업분 — 본 수정 범위 밖.
6. **오프라인 감지 배너**(HIGH H-2): 후속 작업.
