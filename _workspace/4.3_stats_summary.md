## 아키텍처 / 설계 결정

### 1) 순수 유틸 분리 (`stats.js`)
- `document`, `chrome.*`, `Math.random` 미사용 → 동일 입력 → 동일 출력.
- `toNumbers()` 로 `null/NaN/undefined/비숫자` 정리 후 계산. 빈 배열은 **0 반환**(예외 금지) — UI 렌더 중 안전.
- `stdev`는 표본 표준편차(분모 `n-1`), 모표준편차는 `stdevPopulation` 별도 제공.
- `distribute(arr, n)` 는 바차트 히스토그램 입력용으로 `{rangeStart, rangeEnd, count, label}` 반환.

### 2) SVG 직접 생성 (`bar-chart.js`)
- `const SVG_NS = 'http://www.w3.org/2000/svg'` + `document.createElementNS(SVG_NS, tag)` 전용 헬퍼 `svgEl()`.
- 텍스트는 `safeText()`(dom-safe.js) 경유 — XSS 차단.
- 반응형: `width="100%" height="200" viewBox="0 0 W H" preserveAspectRatio="none"`. 부모 컨테이너 너비에 가로 스케일, 세로는 고정.
- 바 색상 자동: 상위 70% good(녹색) / 40% fair(황) / 그 외 poor(적). `barColor` 옵션으로 오버라이드 가능.
- 빈 입력 시 "데이터 없음" 텍스트 노드 삽입(무음 실패 금지).
- 호버 툴팁: 각 `<rect>` 내부 `<title>` 자식으로 네이티브 제공 → JS 이벤트 없이 접근성 확보.

### 3) 워드클라우드 CSS Grid (`word-cloud.js`)
- 5-bucket font-size: `[12, 14, 17, 20, 24]` px.
- 5-bucket color: `['#94a3b8', '#64748b', '#475569', '#2563eb', '#1d4ed8']` — 가중치가 높을수록 진한 파랑.
- Bucket 계산: `Math.floor((w - min) / ((max-min)/5))`. 모든 weight 동일 시 중간 bucket(2)로 통일.
- 레이아웃: `display: grid; grid-template-columns: repeat(auto-fill, minmax(72px, 1fr))` — 사용자 지정 **CSS Grid 배치** 요건 충족.
- 외부 라이브러리 의존 0.

### 4) 비교 카드 (`comparison-card.js`)
- `METRICS` 배열에 지표 정의(`{key, label, higherIsBetter, format}`) — 정책 변경 시 단일 지점.
- 지표: 종합 점수 / 글자 수 / 이미지 수 / 단락 수 / 평균 문장 길이.
- `computeDelta()` — `higherIsBetter` 에 따라 양/음 방향이 "좋음(녹색)" 인지 자동 판정. 표시: `▲ 5.2` / `▼ 120` / `±0`.
- 양쪽 모두 null 시 "비교할 데이터가 없습니다" 폴백.

### 5) 벤치마크 통계 섹션 (`benchmark-stats-section.js`)
- **호출자 책임 원칙**: `benchmark-repo` 를 직접 import 하지 **않음**. `{ posts, loading?, error? }` prop 으로 받는 순수 렌더러. 테스트 용이 + repo scope 제한.
- `metrics` JSONB 우선, 없으면 `seo_score`/`word_count`/`image_count` 테이블 컬럼 폴백.
- 요약 3 카드: 평균 점수(+중앙값·σ) / 평균 글자수(+min~max) / 평균 이미지수(+max).
- 점수 분포는 `distribute()` 로 5 bucket 히스토그램 생성 후 `createBarChart` 주입.
- 상위 키워드는 모든 post 의 `topKeywords` flatten → Map 카운트 → 상위 30 → `createWordCloud`.
- `renderBenchmarkStatsInto(container, options)` 헬퍼로 재렌더 지원.
