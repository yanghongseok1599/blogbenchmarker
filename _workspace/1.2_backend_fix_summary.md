## 추가 수정 (체크리스트 외 개선)

- **`profiles.updated_at`** 추가 + `set_updated_at()` plpgsql 함수 + `trg_profiles_set_updated_at` BEFORE UPDATE 트리거 (QA B-9 MEDIUM 이기도 함). HIGH 수정의 연장선에서 같은 파일을 건드리는 김에 반영. moddatetime 확장 의존 없음.
- **Idempotent 패치 블록** (001, 005): 이미 이전 스키마로 DB 를 push 한 환경을 위해 `ALTER TABLE ADD COLUMN IF NOT EXISTS` / DO 블록 기반 RENAME COLUMN / 조건부 CHECK·UNIQUE 추가를 병행. `supabase db reset` 흐름과 기존 배포 두 케이스 모두 안전.
