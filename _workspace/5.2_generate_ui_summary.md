## 옵션 UI ↔ Edge Function 계약 매핑

사용자 요구사항(slider 0~100, select short/medium/long, checkbox)과 Edge Function 계약(enum 3단계)이 불일치 → **tab 레이어에서 변환**:

| UI 입력 | 저장 값 | Edge Function 전송값 |
|---|---|---|
| originality slider | 0~100 | `preserve` (0~33) / `remix` (34~66) / `creative` (67~100) |
| length select | `short` / `medium` / `long` | `short` / **`normal`** (medium→normal) / `long` |
| extraNotes textarea | 문자열 (max 500자) | `options.extraNotes` |
| useLearning checkbox | true/false | true 시 `learning_data` 최근 3건의 `content_json.body` 를 `learningRefs[]` 로 삽입 |
| topic input | 문자열 (max 500자) | `topic` (필수) |

## 동작 흐름

```text
1) mountGenerateTab(panelRoot)
   └─ DOM 조립 (createEl, innerHTML 0건)
   └─ form submit 핸들러 + range input 이벤트 바인딩

2) 사용자 "생성" 클릭
   └─ handleSubmit
       ├─ topic 필수 검증
       ├─ buildOptions()
       │   ├─ mapOriginality(0~100) → 'preserve' | 'remix' | 'creative'
       │   ├─ mapLength(short|medium|long) → 'short' | 'normal' | 'long'
       │   └─ useLearning 체크 시 → fetchLearningRefs() → supabase.from('learning_data')
       ├─ setBusy(true)  [모든 컨트롤 disabled + "생성 중..." 라벨]
       ├─ showStatus('AI 가 글을 작성 중입니다...', 'info')
       ├─ sendGenerateMessage({ topic, options })
       │     ↓ chrome.runtime.sendMessage({ action: 'generate.content', payload })
       │     ↓ service-worker → generateHandler.content
       │        ├─ getSession() → 없으면 '로그인이 필요합니다.'
       │        ├─ normalizePayload → LIMITS 클램프 + ALLOWED 화이트리스트
       │        ├─ supabase.functions.invoke('generate-content', { body })
       │        │   (SDK 가 Authorization Bearer 자동 첨부)
       │        ├─ 에러 코드 → mapEdgeError → 한국어 메시지 throw
       │        └─ 성공 data 반환
       ├─ renderResult(topic, data)
       │     └─ createGenerateResultCard({ content, quota, onCopy, onSaveLearning })
       └─ setBusy(false)

3) 결과 카드 상호작용
   ├─ 복사 → copyText(editor.value) → clipboard-api 또는 execCommand 폴백
   └─ 학습 데이터로 저장 → saveAsLearning(topic, editor.value)
         └─ supabase.from('learning_data').insert({ user_id, content_json, meta })
```

## Edge Function 계약 연동 (contract §1)

- URL 대신 SDK 의 `supabase.functions.invoke(FUNCTION_NAME, { body })` 사용 → Authorization 헤더 자동, CORS preflight 자동.
- 응답 봉투 `{ ok: true, data }` / `{ ok: false, error: { code, message, details } }` 를 그대로 해석.
- 에러 코드 10종(invalid_input / missing_authorization / invalid_token / profile_not_found / rate_limit / quota_exceeded / invalid_key / missing_key / upstream_error / invalid_response / server_misconfig) 한국어 매핑. 알 수 없는 코드는 `error.message` fallback.
- `error.details` 는 그대로 전달 (rate_limit/quota_exceeded 시 `{ minuteCount, dailyCount, dailyQuota, minuteLimit }`). 향후 UI 가 이를 활용해 배지/프로그레스 표시 가능.

## 보안 체크

- ✅ Gemini API 키는 Edge Function 에서만 처리, 확장에 노출 없음
- ✅ 생성 결과 본문은 `textarea.value` 로만 설정 — HTML 으로 해석될 경로 없음
- ✅ 학습 참조(learningRefs) 로딩 시 user_id 필터 + RLS 로 타인 데이터 접근 불가
- ✅ 저장 시 `content_json.body` 필드에 텍스트만 저장 — HTML 저장 시에도 읽기 쪽에서 textContent 로 처리하므로 XSS 경로 없음
- ✅ 복사 폴백의 `<textarea>` 는 `readonly` + `value` 프로퍼티만 사용 — HTML 해석 차단

## 사용 예제

### 탭 단독 마운트 (테스트)

```js
import { mountGenerateTab } from './tabs/generate-tab.js'
const panel = document.querySelector('[data-panel="generate"]')
mountGenerateTab(panel)
```

### 복사 유틸 단독

```js
import { copyText } from '../lib/utils/clipboard.js'
const res = await copyText('안녕하세요')
if (!res.ok) alert(res.error)  // 친화 메시지
```

### 결과 카드 단독

```js
import { createGenerateResultCard } from '../components/generate-result-card.js'
const card = createGenerateResultCard({
  topic: '테스트',
  content: '초안...',
  quota: { minuteCount: 1, dailyCount: 2, dailyQuota: 3, minuteLimit: 10 },
  onCopy: (text) => copyText(text),
  onSaveLearning: (text) => saveAsLearning('테스트', text),
})
document.body.appendChild(card)
```
