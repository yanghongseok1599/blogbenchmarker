## 1. 설치

1. Chrome 브라우저를 열고 아래 링크를 방문합니다.
   - Chrome Web Store: {{WEBSTORE_URL}}
2. **Chrome에 추가** 버튼을 클릭하고 권한 요청 창에서 **확장 프로그램 추가** 를 선택합니다.
3. 설치가 완료되면 브라우저 우측 상단 툴바에 BLOG BenchMarker 아이콘(파란 B)이 표시됩니다. 아이콘이 보이지 않으면 퍼즐 모양 확장 버튼을 눌러 고정하세요.

> _스크린샷 위치:_ `![설치 완료](docs/img/01-install.png)`

### 요구사항

- Chrome 116 이상 또는 호환 Chromium 브라우저(Edge, Whale 등).
- 네이버 블로그 사용자 계정(필수는 아니나 분석 대상이 필요).
- BLOG BenchMarker 계정(이메일 또는 Google).

## 2. 회원가입과 로그인

### 2.1 회원가입

1. 툴바의 BLOG BenchMarker 아이콘을 클릭해 사이드패널을 엽니다.
2. 최초 실행 시 로그인 페이지가 새 탭으로 열립니다. **회원가입** 링크를 눌러 이동합니다.
3. 이메일, 비밀번호(8~72자, 영문+숫자 포함), 비밀번호 확인을 입력합니다.
4. 이용약관·개인정보 처리방침 동의 체크박스를 선택하고 **가입하기** 를 누릅니다.
5. 입력한 이메일 주소로 인증 메일이 발송됩니다. 메일의 **계정 확인** 링크를 클릭하면 가입이 완료됩니다.

> _스크린샷 위치:_ `![회원가입](docs/img/02-signup.png)`

### 2.2 Google 로그인

1. 로그인 페이지의 **Google로 계속하기** 버튼을 누릅니다.
2. Google 계정 선택 → 권한 동의(이메일·프로필) → BLOG BenchMarker 로 자동 복귀합니다.

> _스크린샷 위치:_ `![Google 로그인](docs/img/03-google.png)`

### 2.3 비밀번호 재설정

1. 로그인 페이지에서 **비밀번호 찾기** 를 누릅니다.
2. 가입한 이메일을 입력하고 **재설정 메일 보내기** 를 누르면 재설정 링크가 발송됩니다.
3. 메일의 링크를 클릭해 새 비밀번호를 설정합니다.

## 3. SEO 분석 사용하기

### 3.1 글 페이지에서 분석

1. 네이버 블로그 글(예: `https://blog.naver.com/aaa/222...`)을 엽니다.
2. 사이드패널 **분석** 탭을 열고 **이 글 분석하기** 버튼을 클릭합니다.
3. 약 1~3초 후 SEO 점수(100점 만점)와 5개 섹션(제목 최적화·콘텐츠·후킹·가독성·키워드 밀도), 첫 문장 후킹 유형, 개선 제안이 표시됩니다.

> _스크린샷 위치:_ `![분석 결과](docs/img/04-analyze-result.png)`

### 3.2 글쓰기 페이지의 실시간 사이드바

1. 네이버 블로그 글쓰기 페이지(`PostWriteForm.naver` 등)로 이동하면 우측에 분석 사이드바가 자동 표시됩니다. 필요 없으면 **Alt+B** 로 토글할 수 있습니다.
2. 글을 작성하는 동안 800ms 간격으로 자동 분석되어 점수가 갱신됩니다.

### 3.3 점수 해석

| 섹션 | 만점 | 핵심 포인트 |
|---|---:|---|
| 제목 최적화 | 20 | 20~40자, 키워드 포함, 숫자·괄호 등 시각 요소 |
| 콘텐츠 구조 | 30 | 문단 3~15개, 이미지 3~10개, 첫 문단 FIRE, 신뢰 요소 |
| 후킹 | 15 | 첫 문장의 강도(질문/감탄/통계/스토리텔링/인사) |
| 가독성 | 20 | 평균 문장 길이, 문장 유형 분포, 이모지 남용 |
| 키워드 밀도 | 15 | 상위 키워드의 제목 매칭도, 최상위 밀도 % |

> 점수는 절대 기준이 아니라 **상대적 개선 힌트** 로 활용해 주세요. 검색 결과 순위를 보장하지 않습니다.

## 4. 경쟁 블로그 벤치마킹

### 4.1 경쟁 블로거 등록

1. 사이드패널 **벤치마크** 탭을 엽니다.
2. 상단 입력창에 경쟁 블로그 URL 을 붙여넣고 **추가** 를 누릅니다.
   - 예: `https://blog.naver.com/competitor`
3. 최대 10개(PRO) / 1개(FREE)까지 등록 가능합니다.

> _스크린샷 위치:_ `![벤치마크 등록](docs/img/05-benchmark-add.png)`

### 4.2 자동 수집

- 등록 직후 최근 글 목록을 자동 수집합니다(30분 간격 재동기화).
- 수집된 글은 벤치마크 탭에서 **점수·글자수·이미지수** 로 정렬·필터할 수 있습니다.

### 4.3 통계 보기

- 탭 하단의 **통계** 섹션에서 평균 점수·글자수·이미지수, SEO 점수 분포(5 bucket 바 차트), 자주 등장한 키워드(워드클라우드)를 확인합니다.
- PRO 플랜은 다기기 Realtime 동기화가 적용되어 PC와 노트북에서 즉시 동일한 데이터를 봅니다.

## 5. AI 초안 생성

### 5.1 기본 흐름

1. 사이드패널 **생성** 탭을 엽니다.
2. 프롬프트 텍스트박스에 작성하고 싶은 글의 주제·핵심 메시지를 입력합니다.
3. 옵션을 선택합니다.
   - **길이:** 80% / 100% / 120%
   - **모드:** 구조 유지 / 재해석 / 창작
   - **톤·역할·포함 키워드(선택)**
4. **초안 생성** 버튼을 누르면 5~15초 내로 결과가 표시됩니다.
5. 상단 **복사** 버튼으로 클립보드에 복사한 뒤 네이버 글쓰기 페이지에 붙여넣습니다.

> _스크린샷 위치:_ `![AI 초안](docs/img/06-generate.png)`

### 5.2 사용량 제한

| 플랜 | 일일 생성 한도 | 월간 한도 |
|---|---:|---:|
| FREE | 3회 | 90회 |
| PRO | 30회 | 500회 |

FREE 회원은 마이페이지에서 본인 Google Gemini API 키(BYOK) 를 등록해 시스템 한도를 초과할 수 있습니다.

### 5.3 주의 사항

- AI 생성 결과는 **사실관계 검증 없이 그대로 발행하지 마세요.** 인용·통계·인명 등은 직접 확인 후 반영합니다.
- 저작권 침해 방지를 위해 경쟁 글의 **문장 단위 복제** 가 아닌 **구조·톤 학습** 으로만 활용합니다.

## 6. 학습 엔진 활용

### 6.1 학습 데이터 저장

1. 분석 탭에서 결과 하단의 **분석 + 학습** 토글을 켭니다.
2. 저장된 글은 **학습** 탭에서 목록으로 확인할 수 있으며, 체크박스로 개별 삭제 가능합니다.
3. FREE 50개 / PRO 무제한.

### 6.2 AI 생성 시 컨텍스트 주입

- 생성 탭 옵션에 **학습 데이터 사용** 체크박스가 활성화되어 있으면, 가장 최근 5개 학습 글의 구조·톤·키워드가 프롬프트에 컨텍스트로 포함됩니다.
- 학습 데이터가 많을수록 본인 문체에 가까운 초안이 생성됩니다.

## 7. YouTube 영상 → 블로그 변환

1. 사이드패널 **생성** 탭의 **YouTube** 하위 탭을 엽니다.
2. YouTube 동영상 URL 을 입력하고 **자막 가져오기** 를 누릅니다(자막이 있는 영상만 지원).
3. 자막이 추출되면 **블로그 형식으로 변환** 을 누릅니다. 제목·소제목·단락 구조의 초안이 생성됩니다.
4. FREE 일 3건 / PRO 일 30건 한도.

## 8. 글쓰기 보조 도구

사이드패널 **도구** 탭에서 4개 도구를 사용할 수 있습니다.

### 8.1 글자수 카운터

- 입력 상자에 글을 붙여넣으면 실시간으로 6가지 지표(글자 수 공백 포함/제외, 단어·문장·문단 수, 예상 읽기 시간)가 갱신됩니다.
- 네이버 블로그는 **공백 제외 글자수** 를 기준으로 사용합니다(분당 읽기 속도는 한국어 500자 기준).

### 8.2 뽀모도로 타이머

- 기본 25분 집중 / 5분 휴식. 설정에서 변경 가능.
- **시작 → 일시정지 → 리셋** 3개 버튼.
- 사이드패널을 닫아도 background 에서 타이머가 계속 진행되며, 종료 시 브라우저 알림이 울립니다.

### 8.3 금칙어 검사

- 관리자가 등록한 전역 금칙어 + 사용자 정의 단어 모두 검사합니다.
- 사용자 단어는 쉼표로 구분해 한 번에 여러 개 추가할 수 있습니다.
- 매칭된 부분은 노란 배경으로 하이라이트 되며, 툴팁으로 어떤 단어에 걸렸는지 확인할 수 있습니다.

### 8.4 스크린샷

- **영역 캡처:** 현재 네이버 블로그 탭 위에 반투명 오버레이가 뜹니다. 드래그로 영역을 지정하세요. ESC 로 취소.
- **전체 캡처:** 현재 탭의 보이는 영역 전체를 PNG 로 저장합니다.
- 미리보기 하단의 **저장** 버튼으로 로컬에 다운로드합니다.

> _스크린샷 위치:_ `![도구 탭](docs/img/07-tools.png)`

## 9. 마이페이지 · 플랜 관리

### 9.1 마이페이지 열기

- 사이드패널 **마이** 탭 하단 **전체 보기** 버튼 → 독립 탭으로 마이페이지가 열립니다.

### 9.2 확인할 수 있는 정보

- 프로필(이메일·표시 이름·가입일)
- 현재 플랜 및 상태·만료일
- 이번 달 사용량 게이지(한도 대비 %)
- 최근 30일 기능별 사용 횟수·토큰 소비량

### 9.3 PRO 업그레이드

1. 마이페이지 또는 사이드패널 만료 배너의 **지금 연장** / **플랜 업그레이드** 를 누릅니다.
2. 토스페이먼츠 또는 포트원 결제창이 열립니다.
3. 결제 완료 후 즉시 플랜이 반영됩니다(최대 1분 소요).

### 9.4 만료 알림

- 만료 3일 전: 노란색 배너
- 만료 1일 전: 빨간색 배너
- 만료 후: 회색 배너 + 기능 제한

배너의 **닫기** 버튼은 해당 만료일 기준으로만 숨겨집니다. 재결제 시 새 배너가 자동으로 나타납니다.

### 9.5 계정 삭제

- 마이페이지 하단 **계정 삭제** 링크 또는 {{SUPPORT_EMAIL}} 로 요청.
- 삭제 시 모든 분석·학습·결제 이력(법정 보관 기간 외)이 즉시 파기됩니다.

## 10. 문제 해결

일반적인 문제는 [FAQ 문서](./FAQ.md) 를 먼저 확인해 주세요. 해결되지 않으면 {{SUPPORT_EMAIL}} 로 다음 정보를 포함해 문의해 주세요.

- 확장프로그램 버전 (마이페이지 하단에 표시)
- Chrome 버전 (`chrome://settings/help`)
- 발생한 문제의 재현 절차
- (가능하면) 스크린샷 또는 콘솔 로그(`Ctrl+Shift+J` → Console 탭 내용)

**민감 정보(이메일, 비밀번호, 결제 정보)는 절대 첨부하지 마세요.** 고객센터 담당자가 요청하더라도 본인 확인은 공식 이메일 템플릿으로만 진행됩니다.
