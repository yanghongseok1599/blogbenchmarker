## 2. 스크린샷 5종 구성

### Shot 1 — 분석 탭 (메인)

**화면:** 네이버 블로그 글 페이지를 연 상태에서 사이드패널 "분석" 탭이 SEO 점수(예: 82/100), 5개 섹션 진행바, 추천 개선 제안 3건을 보여주는 화면.

**강조 포인트:** 100점 만점 원형 점수 카드, 진행 바의 색 구분(good/fair/poor), 한국어 개선 제안 목록.

**캡션:**

| 언어 | 내용 |
|---|---|
| ko | 네이버 블로그 글의 SEO 점수를 5개 섹션으로 자동 분석합니다 |
| en | Automatically score your Naver blog post across five SEO dimensions |
| ja | NaverブログのSEOスコアを5つのセクションで自動分析します |

---

### Shot 2 — 벤치마크 탭 (경쟁 블로그 통계)

**화면:** 벤치마크 탭에서 3~5개의 경쟁 블로그를 등록한 상태. 평균 글자수/이미지수/점수 카드, 점수 분포 SVG 바 차트, 키워드 워드클라우드가 보이는 구성.

**강조 포인트:** 바 차트의 5 bucket 색상, 워드클라우드의 bucket별 폰트 크기 차이.

**캡션:**

| 언어 | 내용 |
|---|---|
| ko | 경쟁 블로그 평균 지표와 키워드를 한눈에 비교합니다 |
| en | Compare key metrics and trending keywords across competitor blogs |
| ja | 競合ブログの平均指標とキーワードを一目で比較できます |

---

### Shot 3 — 생성 탭 (AI 초안)

**화면:** 생성 탭에서 옵션(길이 100%, 모드 "구조 유지")을 선택한 뒤 생성된 초안이 우측에 표시되고, "복사" 버튼이 눌린 직후의 상태.

**강조 포인트:** 옵션 패널의 라디오 버튼, 생성 결과의 단락 구조, "복사했습니다" 토스트.

**캡션:**

| 언어 | 내용 |
|---|---|
| ko | 학습된 데이터를 바탕으로 내 문체에 맞는 초안을 생성합니다 |
| en | Generate a draft in your own voice using your learned writing style |
| ja | 学習データをもとに、自分の文体に合った下書きを生成します |

---

### Shot 4 — 마이페이지 (플랜·사용량 대시보드)

**화면:** 마이페이지 전체 페이지 또는 사이드패널 "마이" 탭. 프로필 카드 + PRO 뱃지 + 이번 달 사용량 게이지(예: 45%) + 최근 30일 기능별 사용 내역 표.

**강조 포인트:** 게이지 색상(good 상태), 기능별 표의 한국어 라벨, 만료일(D-14 등).

**캡션:**

| 언어 | 내용 |
|---|---|
| ko | 플랜 상태와 이번 달 사용량을 마이페이지에서 한 번에 확인 |
| en | Review your plan status and monthly usage in one place |
| ja | プランの状態と今月の利用状況をマイページで一括確認 |

---

### Shot 5 — 부가 도구 탭 (4 in 1)

**화면:** 도구 탭에서 글자수 카운터·뽀모도로(진행 중)·금칙어 검사·스크린샷 카드가 순서대로 보이는 화면. 뽀모도로는 "집중 17:42" 등 타이머 동작 중.

**강조 포인트:** 4개 카드 레이아웃, 카운터의 통계 그리드, 뽀모도로 타이머 숫자, 금칙어 하이라이트 `<mark>` 색상.

**캡션:**

| 언어 | 내용 |
|---|---|
| ko | 글자수·뽀모도로·금칙어·스크린샷까지 글쓰기를 돕는 도구 모음 |
| en | Writing aids — character counter, pomodoro, profanity filter, screenshot |
| ja | 文字数・ポモドーロ・禁止語・スクリーンショットまで執筆を支援するツール |

## 3. 촬영 환경 및 체크리스트

### 3.1 브라우저 설정

- **창 크기:** 1440×900 (디바이스 픽셀비율 1.0 기준) — 사이드패널 너비 표준 360px 확보.
- **확대/축소:** 100% 고정 (Cmd/Ctrl + 0).
- **탭바 및 북마크 바:** 숨기기 권장(Chrome `--hide-crashed-bubble`, 북마크 바 비활성화).
- **프로필:** 개인정보가 노출되지 않는 빈 프로필 사용. Gmail·노출된 탭 제거.
- **언어:** Chrome 언어 "한국어" 1순위.

### 3.2 샘플 데이터

- **블로그 글:** 공개된 샘플 콘텐츠를 새로 작성. 실제 이용자 글은 **사전 서면 동의 없이 사용 금지.**
- **경쟁 블로그:** 더미 이름("데모 블로거 A/B/C") 으로 등록.
- **사용량:** 데모용 PRO 계정에서 생성 30회·분석 120회 수준으로 시뮬레이트.

### 3.3 촬영 체크리스트

- [ ] 개인 이메일·알림·실시간 시간 등 민감 정보 제거(데모 이메일 사용)
- [ ] 한국어 UI 스펠링·줄바꿈 점검(타이포 없음)
- [ ] 만료일·결제 금액 등 실제 수치 노출 시 더미로 교체
- [ ] 네이버 저작권 이미지 포함 여부 확인 — 포함된 경우 캡션에서 가림 또는 교체
- [ ] Lighthouse/DevTools 패널 등 개발자 창이 열려 있지 않음
- [ ] 다크 모드 혼재 없음(라이트 통일)

## 4. 후처리 가이드

- **크롭:** 정확히 1280×800 으로 크롭. 여백이 생기면 브랜드 색 `#2563eb` 또는 `#f8fafc` 로 채움.
- **주석 텍스트:** 장식 문구 삽입 시 Noto Sans KR Bold 24~32pt, 키워드 1행 이내.
- **하이라이트 박스:** 얇은 `#2563eb` 테두리(2px) + 2% 투명 오버레이 허용.
- **블러:** 개인정보 노출부에만 8~12px 가우시안 블러.
- **기기 프레임:** Chrome 자체 UI 를 포함하되, 허구의 브라우저 프레임은 사용하지 않는다(스토어 정책).

## 6. 금지 사항

- 실제 이용자의 블로그 글·사진을 동의 없이 사용
- 경쟁 확장 프로그램 또는 제3자 브랜드 로고 포함
- 실제 결제 금액·카드번호·영수증 노출
- "보장", "무조건 1위", "수익 확정" 등 과장 문구 (스토어 정책 위반)
- Chrome UI 또는 네이버 UI 의 왜곡·합성 (reality manipulation 위반)
- 정치·종교·성인물 관련 이미지

위반 시 스토어 심사 반려 또는 등록 후 강제 제거 대상이 됩니다.
