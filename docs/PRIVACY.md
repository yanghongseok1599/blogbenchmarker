## 3. 보관 기간 및 파기

- **회원 탈퇴 시:** `auth.users` 삭제와 동시에 FK CASCADE 로 프로필, 학습 데이터, 벤치마크 목록, 사용량 로그, 결제 이력이 즉시 삭제됩니다. 물리적 파기까지 최대 30일 소요될 수 있습니다(데이터베이스 백업 회수 주기).
- **결제 기록:** 「전자상거래 등에서의 소비자보호에 관한 법률」에 따라 5년간 보관(결제·환불 내역 한정). 해당 기간 동안은 탈퇴 후에도 최소 식별자(익명화된 결제 ID)로 보존됩니다.
- **사용량 로그:** 분쟁·오남용 대응을 위해 최대 12개월 보관 후 자동 삭제됩니다.
- **쿠키/로컬 스토리지:** 이용자가 확장프로그램을 제거하거나 브라우저 캐시를 삭제하면 즉시 삭제됩니다.

**파기 방식:** DB 레코드는 `DELETE` 쿼리로 논리적 삭제, 이후 백업 회수 과정에서 물리적 파기. 로컬 파일은 브라우저 API(`chrome.storage.*.clear`)로 즉시 제거.

## 4. 제3자 제공 및 처리 위탁

본 서비스는 이용자 개인정보를 제3자에게 **판매·제공하지 않습니다.** 다만 서비스 제공을 위해 아래의 처리 위탁을 받는 사업자가 있습니다.

| 수탁사 | 처리 업무 | 처리되는 개인정보 | 보관 리전 |
|---|---|---|---|
| Supabase, Inc. (미국) | 계정·데이터 저장, 인증, 실시간 구독 | 이메일, 해시된 비밀번호, 분석·벤치마크 데이터, 사용량 로그 | 이용자 가입 시점의 프로젝트 리전(기본 Asia-Northeast1 또는 us-east-1) |
| Google LLC (미국) — Gemini API | 본문 텍스트를 AI 모델에 전달해 생성 결과 수신 | 생성 요청 시 전달되는 블로그 본문 및 제목 | Google 정책에 따름 |
| Google LLC — OAuth | 이용자가 Google 로그인을 선택한 경우 식별자 확인 | 이메일, 고유 식별자 | — |
| 토스페이먼츠(주) 또는 (주)포트원 | 결제 처리 및 웹훅 검증 | 결제 금액, 주문 번호, 결제 수단 메타(카드번호 원본은 처리하지 않음) | 국내 |

**국외 이전 고지:** Supabase 및 Google 서비스 이용을 위해 이용자 데이터의 일부가 대한민국 외 국가로 전송될 수 있습니다. 계정 생성 시 이용자는 이에 대한 고지를 확인하고 동의하게 됩니다.

**Google Gemini 전송 데이터:**

- AI 글 생성 요청 시에만 본문·제목이 전달되며, 분석 요청(SEO 점수 계산)은 확장 내부에서 수행되어 전송되지 않습니다.
- Google 이 전송된 데이터를 모델 학습에 사용하지 않도록 API 옵션을 설정합니다(2026-04-14 기준 유료 API 기본값).

## 5. 이용자의 권리와 행사 방법

이용자는 언제든지 아래의 권리를 행사할 수 있습니다(GDPR 제15~22조, 국내 개인정보보호법 제35~39조).

| 권리 | 행사 방법 |
|---|---|
| **열람권** | 마이페이지 → 프로필·사용량 대시보드에서 직접 확인 |
| **정정권** | 마이페이지에서 표시 이름 등 수정. 이메일 변경은 {{SUPPORT_EMAIL}} 로 요청 |
| **삭제권(잊혀질 권리)** | 마이페이지 → "계정 삭제" 또는 {{SUPPORT_EMAIL}} 로 요청. 요청 후 30일 이내 처리 |
| **처리 정지권** | {{SUPPORT_EMAIL}} 로 요청 (서비스 이용 일시 중지) |
| **데이터 이동권** | {{SUPPORT_EMAIL}} 로 요청 시 JSON 파일로 본인 데이터 제공 |
| **동의 철회권** | 학습 데이터 저장 동의, 마케팅 수신 동의 등은 설정 화면에서 즉시 철회 |
| **자동화된 의사결정 거부권** | 본 서비스는 프로파일링 기반 자동 결정을 수행하지 않습니다 |

**권리 행사 시 본인 확인:** 이메일 기반 본인 확인 링크 또는 OAuth 재인증을 통해 신원을 확인합니다.

## 6. 쿠키 및 로컬 스토리지 사용

본 서비스는 웹사이트 쿠키를 설정하지 않으며, 크롬 확장프로그램의 `chrome.storage` API 를 사용합니다.

| 저장소 | 용도 | 보관 기간 |
|---|---|---|
| `chrome.storage.local` | Supabase 세션 토큰(자동 로그인), 분석 결과 캐시, 만료 배너 dismiss 기록 | 확장 제거 또는 로그아웃 시까지 |
| `chrome.storage.session` | 일회성 OAuth state 파라미터 | 브라우저 재시작 시 자동 소멸 |
| `chrome.storage.sync` | 사용자 설정(언어 등) — 계정 간 동기화용 | 사용자가 삭제하기 전까지 |

광고·추적용 쿠키는 사용하지 않습니다.

## 7. 안전성 확보 조치

- **전송 구간 암호화:** 모든 외부 통신은 HTTPS(TLS 1.2 이상)로 암호화합니다.
- **저장 구간 암호화:** Supabase PostgreSQL 은 디스크 수준 AES-256 암호화를 적용합니다.
- **접근 통제:** Row Level Security(RLS)로 이용자별 데이터 격리를 데이터베이스 레벨에서 강제합니다.
- **API 키 보호:** Gemini 등 외부 API 키는 서버 측 Edge Function 에만 저장되며, 확장프로그램 번들에 포함되지 않습니다.
- **XSS 방지:** 확장프로그램은 `innerHTML` 대신 안전한 DOM API 만 사용합니다.
- **비밀번호:** 원문은 저장하지 않으며 Supabase Auth 의 bcrypt 해시를 사용합니다.
- **로그 최소화:** 프로덕션 빌드에서는 디버그 로그가 제거됩니다.
