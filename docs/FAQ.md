## 설치와 호환성

### Q1. 어떤 브라우저에서 사용할 수 있나요?

Chrome 116 이상과, 동일한 Chromium 엔진을 사용하는 Microsoft Edge, Naver Whale, Brave 최신 버전을 지원합니다. Safari / Firefox 는 확장 API 호환성 문제로 현재 지원하지 않습니다.

### Q2. 설치 후 사이드패널이 열리지 않아요.

1. 툴바의 확장 아이콘을 **우클릭 → 사이드패널 열기** 를 시도하세요.
2. Chrome 116 이상인지 확인합니다(`chrome://settings/help`).
3. `chrome://extensions` 에서 BLOG BenchMarker 를 비활성화 후 다시 활성화하거나, **서비스 워커** 링크를 눌러 재시작합니다.
4. 그래도 열리지 않으면 확장을 제거 후 다시 설치합니다. 로그인 정보는 계정 기반이므로 삭제되지 않습니다.

## 계정과 로그인

### Q3. 회원가입 이메일이 오지 않아요.

- 스팸/프로모션 폴더를 확인해 주세요.
- 이메일 수신 도메인 차단 정책이 있는 회사 이메일은 발송이 거절될 수 있습니다. 개인 이메일로 재가입하거나 {{SUPPORT_EMAIL}} 로 문의해 주세요.
- 5분 후에도 도착하지 않으면 로그인 페이지의 **비밀번호 찾기** 가 아니라 **재가입(새 이메일)** 을 시도해 보세요(이미 가입된 이메일이면 "이미 가입된 이메일" 메시지가 나타납니다).

### Q4. 로그인이 자꾸 풀립니다.

- 브라우저 쿠키/캐시 자동 삭제 설정이 원인일 수 있습니다. `chrome://settings/content/cookies` 에서 "사이트를 종료할 때 쿠키 및 사이트 데이터 삭제" 옵션을 해제해 주세요.
- 여러 Chrome 프로필을 사용할 경우 프로필마다 별도 로그인이 필요합니다.
- 30일 동안 로그인 갱신이 없으면 보안상 자동 로그아웃됩니다.

### Q5. Google 로그인이 실패합니다.

- 브라우저 팝업 차단 설정을 확인해 주세요.
- Google 계정에 2단계 인증을 사용 중이라면 해당 절차를 완료해야 합니다.
- "보안 검증에 실패했습니다" 메시지가 나오면 페이지를 새로고침한 뒤 다시 시도해 주세요(OAuth state 검증 실패 시 발생).
- 사내망/VPN 에서 Google 도메인이 차단되어 있는지 확인해 주세요.

## 분석과 성능

### Q6. 분석이 너무 느리거나 응답이 없어요.

- SEO 점수 계산(분석)은 확장 내부에서 수행되므로 1~3초 내에 결과가 나와야 정상입니다.
- AI 초안 생성은 Google Gemini 응답 시간에 따라 5~15초 소요됩니다.
- 15초 이상 응답이 없는 경우:
  1. 네트워크 연결 상태를 확인합니다.
  2. `chrome://extensions` 에서 서비스 워커 로그에 "Upstream" 등 오류가 있으면 {{SUPPORT_EMAIL}} 로 캡처와 함께 문의해 주세요.
  3. 분석 대상 글이 매우 긴 경우(10,000자 초과) 분석이 지연될 수 있습니다.

### Q7. SEO 점수가 실제 검색 순위와 달라요.

본 서비스의 점수는 **블로그 작성 시 고려할 요소의 상대적 지표** 로, 네이버 검색 엔진의 실제 랭킹 알고리즘과 동일하지 않습니다. 검색 순위는 노출 기간·사용자 체류 시간·외부 링크·도메인 지수 등 확장에서 측정할 수 없는 요인에 크게 좌우됩니다. 점수를 **개선 방향의 힌트** 로 활용해 주세요.

## 벤치마킹

### Q8. 경쟁 블로그 글 목록이 비어 있어요.

- **블로그가 공개 상태인지** 확인합니다. 이웃공개/비공개 블로그는 수집할 수 없습니다.
- 등록 직후에는 수집까지 최대 30분이 걸릴 수 있습니다. **새로고침** 버튼을 눌러 수동 동기화할 수 있습니다.
- 특수 문자·한글이 포함된 블로그 URL 은 네이버의 정규 URL(`blog.naver.com/ID`) 형식으로 입력해 주세요.

## AI 글 생성

### Q9. AI 결과가 이상하거나 너무 짧습니다.

- **프롬프트 구체화:** "다이어트" 보다는 "30대 여성 대상, 저염식 한식 위주의 다이어트 팁" 처럼 대상·관점·범위를 구체적으로 적어 주세요.
- **길이 설정:** 옵션의 "100%" 또는 "120%" 를 선택하면 긴 초안을 받을 수 있습니다.
- **학습 데이터 활용:** 본인 글 3~5편을 학습 데이터로 저장한 뒤 다시 시도하면 문체가 더 자연스러워집니다.
- **재생성:** 같은 옵션으로 2~3회 재생성하면 품질이 개선되는 경우가 많습니다.

### Q10. BYOK(본인 API 키) 는 어떻게 등록하나요?

1. Google AI Studio(https://aistudio.google.com/app/apikey) 에서 본인 API 키를 발급받습니다.
2. 마이페이지 → **API 키 설정** → 키 입력 → 저장.
3. 키는 서버에 암호화 저장되며, 확장 번들에는 노출되지 않습니다.
4. BYOK 등록 시 시스템 한도 대신 Google 측 요금·쿼터가 적용되며, 요금은 사용자가 Google 에 직접 지불합니다.

## 플랜과 결제

### Q11. PRO 로 업그레이드했는데 한도가 적용되지 않아요.

- 결제 완료 후 최대 1분 이내에 플랜이 반영됩니다. 1분이 지나도 변화가 없으면 사이드패널 **마이** 탭에서 **새로 고침** 을 눌러 주세요.
- 그래도 반영되지 않으면 {{SUPPORT_EMAIL}} 로 결제 주문번호를 포함해 문의해 주세요(결제 웹훅 지연 시 수동 반영).

### Q12. 환불은 어떻게 신청하나요?

- 결제 후 **7일 이내 + 미사용** 인 경우 전액 환불됩니다.
- 환불 신청: {{SUPPORT_EMAIL}} 로 주문번호·이메일·환불 사유를 보내주세요. 영업일 기준 7일 이내 처리됩니다.
- 자세한 환불 정책은 [이용약관](./TERMS.md#53-환불-정책) 을 참고해 주세요.

### Q13. 플랜을 변경(다운그레이드/해지) 하고 싶습니다.

- **해지(다음 결제 주기부터 FREE 전환):** 마이페이지 → **구독 관리** → "해지 예약". 현재 만료일까지는 PRO 혜택을 그대로 이용할 수 있습니다.
- **즉시 해지(환불 포함):** 결제 7일 이내 미사용인 경우에 한해 Q12 의 환불 절차를 따라 신청해 주세요.

## 개인정보와 데이터

### Q14. 내 블로그 글이 서버에 저장되나요?

- **SEO 분석:** 점수 계산은 확장 내부에서 수행되므로 본문이 서버로 전송되지 않습니다.
- **분석 + 학습(선택):** 토글을 켠 경우에만 제목·본문 요약·지표가 Supabase 에 저장됩니다.
- **AI 생성:** 요청 시에만 입력 프롬프트와 학습 컨텍스트가 Supabase Edge Function 을 통해 Google Gemini 에 전달됩니다. Google 의 모델 학습에는 사용되지 않도록 옵션을 설정합니다.
- 자세한 내용: [개인정보 처리방침](./PRIVACY.md)

### Q15. 계정을 완전히 삭제하고 싶어요.

1. 마이페이지 하단의 **계정 삭제** 링크를 클릭합니다(또는 {{SUPPORT_EMAIL}} 로 요청).
2. 이메일 재인증 후 최종 확인 버튼을 누르면 즉시 계정이 삭제됩니다.
3. 학습 데이터·분석 기록·벤치마크·사용량 로그 등 연동된 모든 데이터가 FK CASCADE 로 동시 삭제됩니다.
4. 결제 기록은 관련 법령에 따라 익명화된 상태로 5년간 보관된 후 자동 파기됩니다.
