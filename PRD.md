# BLOG BenchMarker - 제품 요구사항 (PRD)

## 한 줄 정의

네이버 블로그 작성자를 위한 SEO 분석 + 경쟁 블로그 벤치마킹 + AI 글 생성 크롬 확장프로그램.

## 타깃 사용자

- 네이버 블로그를 운영하는 개인 블로거
- 체험단/협찬 후기를 자주 쓰는 블로거
- 블로그 마케팅을 하는 소상공인/마케터

## 핵심 기능

### 1. SEO 분석 (필수)
**위치**: 네이버 블로그 글 페이지 / 글쓰기 페이지

- 점수화 (100점 만점): 첫 문단 품질, 콘텐츠 구조, FIRE 공식, 제목 최적화, 이미지, 신뢰성, 태그
- 글자수, 문단수, 이미지수 카운팅
- 첫 문장 후킹 유형 분석 (질문형/감탄형/통계형/스토리텔링/인사형)
- 어조 분석 (문체, 평균 문장 길이)
- 문장 유형 분포 (서술형/질문형/감탄형 비율)
- 이모지 사용 통계

### 2. 경쟁 블로그 벤치마킹 (신규 강화)
**기존엔 단순 분석만, 신버전은 본격 벤치마킹 도구로**

- 경쟁 블로거 URL 입력 → 최근 글 목록 자동 수집
- 글별 SEO 점수 비교
- 경쟁 블로그 평균 글자수, 문단 구조, 이미지 사용 패턴 통계
- 자주 쓰는 키워드/태그 추출
- 즐겨찾기한 블로거 모니터링 (Realtime 활용)

### 3. AI 글 생성
- 경쟁 글을 학습 데이터로 활용한 글 생성
- 원본 구조 유지 / 재해석 / 창작 3단계 모드
- 글 길이 80% / 100% / 120% 조절
- 추가 요청사항 입력 (역할, 톤, 키워드 포함 등)
- AI가 생성한 글 → 클립보드 복사 → 네이버 글쓰기 페이지에 붙여넣기

### 4. 학습 엔진
- 사용자가 분석한 글들을 데이터로 축적
- 학습 데이터 기반으로 AI 글 생성 품질 개선
- 형태소 분석 (제목, 본문 단어 빈도)

### 5. YouTube → 블로그 변환
- YouTube URL 입력 → 자막 추출 → 블로그 글로 변환
- 일일 사용량 제한 (FREE: 3건, PRO: 무제한)

### 6. 글 작성 보조 도구
- 글자수 카운터 (실시간)
- 뽀모도로 타이머
- 금칙어 체크
- 캡처 도구 (블로그 글 → 이미지)

## 사용자 플로우

### 신규 가입
1. 확장프로그램 설치 → 사이드패널 열기
2. 회원가입 (이메일/비밀번호 또는 Google OAuth)
3. 이메일 인증 (Supabase Auth)
4. FREE 플랜으로 시작

### 일반 사용
1. 네이버 블로그 페이지 방문
2. 사이드패널에서 "이 글 분석하기" 클릭
3. SEO 점수 + 분석 결과 확인
4. (선택) 분석 + 학습 → 학습 데이터에 추가
5. AI 글 생성 → 옵션 설정 → 생성 → 복사

### 벤치마킹
1. 사이드패널에서 "벤치마킹" 탭
2. 경쟁 블로거 URL 추가
3. 자동 수집된 글 목록 확인
4. 평균 통계, 키워드 트렌드 확인

### PRO 구독
1. 마이페이지에서 "PRO 업그레이드" 클릭
2. 결제 (토스/포트원)
3. 자동으로 plan = 'pro' 업데이트
4. 만료 3일 전 알림

## 플랜 정책

| 항목 | FREE | PRO |
|---|---|---|
| SEO 분석 | 무제한 | 무제한 |
| AI 글 생성 | 일 3건 | 무제한 |
| YouTube 변환 | 일 3건 | 무제한 |
| 벤치마킹 모니터링 | 1개 블로그 | 10개 블로그 |
| 학습 데이터 저장 | 50개 | 무제한 |

> FREE는 본인 Gemini API 키 입력 시 무제한 사용 가능 (BYOK 모델)
> PRO는 시스템 제공 API 키 사용

## 신규 추가 기능 (기존 대비)

- ✅ **결제 자동화** (토스페이먼츠/포트원 연동)
- ✅ **만료 알림** (3일 전, 1일 전 사이드패널 배너)
- ✅ **다국어 지원** (한국어, 영어, 일본어)
- ✅ **즐겨찾기 블로거 Realtime 모니터링**
- ✅ **AI 글 피드백 수집** (학습 데이터 품질 개선용)
- ✅ **사용량 대시보드** (마이페이지)

## 제외 기능 (기존에 있었으나 안 만들 것)

- 관리자 페이지의 직접 비밀번호 로그인 → Supabase Auth + RLS로 대체
- 로컬 프로모션 코드 → DB 기반 캠페인 시스템

## 성공 지표

- DAU 100명 (3개월 내)
- PRO 전환율 5% 이상
- 사용자 평균 분석 횟수 일 5회 이상
